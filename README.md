## Running application with npm scrips

```sh
npm install && npm run build
```

### Run as dev

```sh
npm run dev
```

### Run as prod

```sh
npm run deploy
```

### Run docs

```sh
npm run dev:docs
```

or

```sh
npm run start:docs
```
